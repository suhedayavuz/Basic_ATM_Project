# Basic ATM Project

Python diliyle oluşturduğum projede sisteme **yönetici** ve **kullanıcı** olarak giriş yapılmaktadır. 

## Yönetici Olarak:
- **Kullanıcı Ekleme**
- **Kullanıcı Silme**
- **Kullanıcı Şifrelerini Gösterme**
- **Menüye Geri Dönme**

## Kullanıcı Olarak:
- **Para Transferi**
- **Para Çekme**
- **Para Gönderme**
- **Finansal İşlemler**
- **Menüye Geri Dönme**
